# 🦞 OpenClaw(오픈클로) 200% 활용 가이드

안녕하세요, 주인님! 
OpenClaw는 단순한 챗봇이 아니라, 컴퓨터 내부를 직접 제어하는 '진짜 비서'입니다.
현재 상태에서 **바로 할 수 있는 일**과 **준비가 필요한 일**을 정리했습니다.

---

## ✅ 1. 지금 당장 가능합니다 (Ready to Go!)
*별도 설정 없이 명령만 내리면 즉시 실행됩니다.*

### 💬 기억력 (Memory)
*   **"내 차 번호 기억해", "어제 회의 내용 요약해서 저장해"**
*   단기/장기 기억을 관리하며, 나중에 다시 물어보면 정확히 대답합니다. (`MEMORY.md` 활용)

### 📂 파일 관리 (File System)
*   **"바탕화면에 '프로젝트' 폴더 만들어줘", "이 보고서 파일 읽고 요약해줘"**
*   파일 생성, 읽기, 수정, 삭제가 자유롭습니다.
*   복잡한 엑셀/CSV 데이터 정리도 맡겨주세요.

### 💻 코딩 & 터미널 (Coding)
*   **"파이썬으로 환율 계산기 짜줘", "터미널에서 현재 폴더 목록 보여줘"**
*   코드를 직접 작성하고, 파일을 저장하거나 실행할 수 있습니다.

### ❤️ 상태 체크 (Heartbeat)
*   **"30분마다 깨워서 서버 살아있는지 확인해"**
*   주기적으로 시스템 상태를 체크하고 보고합니다.

## ⚠️ 2. 준비물이 필요합니다 (Need Setup)
*간단한 설정이나 API 키가 있으면 더 강력해집니다.*

### 🌐 웹 검색 (Web Search) [키 필요]
*   **"오늘 비트코인 뉴스 검색해줘", "최신 맛집 찾아줘"**
*   👉 **준비물:** Brave Search API 키 (무료)
*   👉 **설정법:** `openclaw configure --section web` 입력 후 키 붙여넣기

### 🖥️ 브라우저 자동화 (Browser Control) [연결 필요]
*   **"구글 문서 열어서 이 내용 타이핑해줘", "특정 사이트 로그인해서 버튼 눌러줘"**
*   👉 **준비물:** 크롬 확장 프로그램 연결 (Connect)
*   👉 **설정법:** 확장 프로그램 아이콘 클릭 -> Port 18789 확인 -> Connect 버튼 클릭

### 📱 메신저 알림 (Messaging) [토큰 필요]
*   **"작업 끝나면 내 텔레그램으로 알려줘"**
*   👉 **준비물:** 텔레그램/슬랙 봇 토큰
*   👉 **설정법:** 설정 파일에 메신저 토큰 추가
